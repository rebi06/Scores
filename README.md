# Scores Program

Javaの基礎練習として作成した、配列とメソッドを使った成績計算プログラムです。  
3人分の点数を入力し、**合計・平均・最高点・最低点**を出力します。

---

## 🧩 学習目的
- 配列にあとから `Scanner` で値を代入する方法を理解する  
- `for` / `拡張for文` の使い方を身につける  
- メソッドを定義して処理を部品化する練習  
- `static` / `non-static` の違いを理解する  

---

## 💻 実行例

```bash
入力:
80
65
90

出力:
合計値は235
平均値は78.33
最高値は90
最低値は65
